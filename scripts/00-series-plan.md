# Video Series Plan

## Goal

講座資料をベースにしつつ、個人の YouTube 向けに見やすい Tutorial シリーズへ再構成する。

## Basic Policy

- 本編は「最終成果物を早く見せる」
- 途中で詰まりやすい箇所は本編で軽く触れ、詳しい回避策は概要欄や補足へ回す
- UI やモデル名など変わりやすい情報は言い切りすぎない
- 画面操作が長い場面はカットし、必要ならテロップ補足にする

## Tentative Episodes

1. Codex でサイトを作って GitHub Pages に公開する
2. CSS と JavaScript で見た目と動きを足す
3. Tips、画像添付、参考デザイン再現

## Reuse From Course Notes

- `01-02`: 前提説明から必要部分だけ抜き出す
- `03-04`: 本編の中心
- `05`: 続編や後半パート向き
- `06`: 補足回や Shorts 向き

## Open Notes

- 本編を 1 本でまとめるか、前後編に分けるかはあとで判断
- 音声はアドリブ中心でもよいが、冒頭 30 秒と締めは先に決めておく
