# Shot List

## Main Tutorial

- 完成したページの冒頭チラ見せ
- GitHub で新規リポジトリを作る
- Codespaces を起動する
- Codex CLI を開く
- プロンプトを入力する
- `index.html` ができる
- `python3 -m http.server 5500` で確認する
- push する
- Pages を有効化する
- 公開 URL を開く

## Optional Inserts

- タブ切り替えの説明
- `git add -> commit -> push` の図解
- Actions でデプロイ確認

## Cuttable Parts

- Codespaces の待ち時間
- push 待ち
- Pages 反映待ち
- 同じ説明を繰り返す部分
