# 02 Design Upgrade

## Goal

素の HTML だけのページに、CSS と JavaScript を足して見た目と動きを良くする。

## Core Message

「Codex はゼロから作るだけでなく、見た目の改善やアイデア出しにも相性がいい。」

## Draft Flow

1. 前回のページを開く
2. コンテンツを少し追加する
3. CSS を追加する
4. デザインの方向性を変えて遊ぶ
5. JavaScript で軽い動きを足す
6. push して反映確認

## Candidate Prompts

- 自己紹介ページにしたいので、質問しながら内容を作ってください
- CSS ファイルを追加して、見た目を良くしてください
- レトロゲーム風のデザインにしてください
- タイピング風アニメーションを追加してください

## Notes

- デザイン例は 2 つくらいに絞る
- 1 回で全部やると長くなるので、好みで前後編に分けてもよい
